# DNS Resolution Explained

## What is DNS?
The Domain Name System (DNS) translates human-readable domain names into IP addresses.

## Step-by-Step Process:

1. User enters a domain (e.g., google.com)
2. Request goes to Local DNS Resolver
3. Resolver queries Root Server
4. Root server directs to TLD server (.com)
5. TLD server directs to Authoritative server
6. Authoritative server returns IP address
7. Browser connects to the server

## Importance
DNS is a critical component of Internet infrastructure, managed globally to ensure stability and accessibility.
